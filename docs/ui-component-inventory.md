# UI Component Inventory

Inventory of reusable UI components used throughout the Dienggo application.

## Core Components

| Component | Description |
| :--- | :--- |
| **Navbar** | Main navigation bar with desktop and mobile support. |
| **ThemeToggle** | Toggle between light and dark modes. |
| **LangToggle** | Toggle between Indonesian and English languages. |
| **SearchBar** | Main search interface for villas and cabins. |
| **SearchBarJeep** | Search interface specialized for Jeep rentals. |
| **SearchBarWisata** | Search interface specialized for Tours (Wisata). |

## Listing and Cards

| Component | Description |
| :--- | :--- |
| **VillaCard** | Card display for Villa listings. |
| **CabinCard** | Card display for Cabin listings. |
| **JeepCard** | Card display for Jeep listings. |
| **WisataCard** | Card display for Tour listings. |
| **CardSkeleton** | Loading state placeholder for cards. |
| **FacilityBadge** | Small badge for displaying entity features/facilities. |

## Layout and Data Display

| Component | Description |
| :--- | :--- |
| **BentoGrid** | Modern grid layout for highlighting featured content. |
| **PromoBanner** | Promotional banner for discounts or events. |
| **VideoSlider** | Slider for displaying video content (Reels). |
| **TourTimeline** | Timeline display for tour itineraries. |
| **FilterTabs** | Category selection tabs. |
| **StickyBookingBar** | Fixed bar at the bottom for quick booking actions on mobile/tablet. |
| **EmptyState** | Display when no data matches search or filters. |

## Admin Components

| Component | Description |
| :--- | :--- |
| **Sidebar** | Navigation sidebar for the admin dashboard. |
| **DataTable** | Reusable table with sorting and actions. |
| **ImageUploader** | Component for handling multi-image uploads to S3. |
| **FormField** | Standardized input wrapper for admin forms. |
| **StatsCard** | Quick overview card for dashboard metrics. |
| **StatusBadge** | Color-coded status indicator. |
| **DeleteDialog** | Confirmation dialog for destructive actions. |
